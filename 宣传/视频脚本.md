# TermKit 视频脚本

---

## 视频信息

- **时长目标**：3-5 分钟
- **风格**：屏幕录制 + 画外音解说
- **分辨率**：1920x1080 或 2560x1440（Retina 录制后缩放）

---

## 开场（0:00 - 0:30）

**画面**：终端界面，手动输入 `cd ~/Projects/MyApp && claude --resume`

**旁白**：

> 用 AI CLI 写代码的人，每天打开终端第一件事就是：cd 到项目目录，启动 Claude Code 或者 Gemini CLI，继续昨天的对话。
>
> 两行命令，不难，但是一天重复几十次就很烦。
>
> 今天介绍一个我自己写的工具——TermKit。它能把这些操作压缩到一次按键里。

---

## 核心演示（0:30 - 1:30）

**画面**：展示完整操作流程

**旁白**：

> 看我操作。我长按 Command 键，菜单弹出来了。

**操作**：长按 ⌘，菜单弹出

> 按 1 选项目文件夹。

**操作**：按数字键 1，进入文件夹列表，选一个项目

> 再按 1 选 Claude Code，然后按 2 选 Resume。

**操作**：按数字键选 CLI 和动作

> 现在松开 Command。

**操作**：松开 ⌘，命令自动粘贴到终端

> 命令已经粘贴好了：cd 到项目目录，然后 claude --resume。全程没打一个字。

**画面**：终端中显示完整的粘贴命令

---

## 功能介绍一：多种导航方式（1:30 - 2:00）

**旁白**：

> 菜单支持多种操作方式。数字键 1 到 9 直接选对应的项，方向键上下移动，左右键切换层级，也可以用鼠标悬停点击。
>
> Esc 取消，松开 Command 确认。

**操作**：分别演示数字键、方向键、鼠标三种方式

---

## 功能介绍二：智能粘贴（2:00 - 2:30）

**旁白**：

> 菜单里有个"粘贴"功能。如果剪贴板有文字，按 V 直接粘贴。
>
> 如果剪贴板有图片——比如你截了一张图想让 AI 看——TermKit 会自动把图片保存到本地，然后粘贴图片路径。

**操作**：
1. 先演示文字粘贴（V 键）
2. 截图后演示图片粘贴，展示路径出现在终端

> 粘贴完成后，剪贴板会自动还原到你之前复制的内容，不影响你的工作流。

---

## 功能介绍三：打开选中路径（2:30 - 3:00）

**旁白**：

> 最新加的功能：打开选中路径。
>
> AI CLI 经常在输出里给出文件路径。以前你得手动复制然后去 Finder 打开。
>
> 现在你只要选中这个路径，长按 Command 弹出菜单，按 L 键。

**操作**：
1. 终端中有一个 Claude Code 输出的文件路径
2. 鼠标选中路径
3. 长按 ⌘ → 按 L
4. 文件在编辑器中打开

> 文件就直接打开了。目录会在 Finder 中打开，代码文件会用默认编辑器打开。带行号的路径也能自动处理。

---

## 功能介绍四：配置（3:00 - 3:30）

**旁白**：

> 所有功能都可以在设置里自定义。

**操作**：打开设置界面，依次切换各标签页

> 文件夹列表可以随意增删，CLI 工具可以添加自定义动作，触发键可以换成 Option 或 Control，响应阈值也能调整。
>
> 白名单功能可以控制只在哪些应用前台时触发菜单，日常用浏览器、微信都不受影响。

## 拍摄/录制备忘

- 录制前清理终端历史和桌面，保持画面干净
- 菜单弹出时适当放慢操作节奏，让观众看清每一步
- L 键打开路径的演示，选一个 .swift 文件效果最直观
- 图片粘贴演示，先截一张有辨识度的图（比如代码截图）
- 确保字体大小够大，视频压缩后小屏也能看清
- 可以加键盘按键可视化工具（如 KeyCastr）让观众看到按了什么键
