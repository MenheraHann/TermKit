## 正文

用 AI 写代码的兄弟们，有没有觉得每天重复做三件事特别烦：

cd 到项目目录 → 启动 Claude Code → 继续昨天的对话

我干脆自己写了个 macOS 小工具：**TermKit**

**长按 ⌘ 弹出菜单 → 选文件夹 → 选 CLI → 松开 ⌘ 命令就粘好了**

全程不用打一个字，键盘不离手。

---

### 核心功能

- 长按 ⌘ 唤出分层菜单，松开确认
- 数字键 1-9 秒选，方向键导航
- 内置 Claude Code / Gemini / Codex / OpenCode 等主流 AI CLI
- 剪贴板有图片？一键保存路径粘到终端，喂给 AI 读
- 命令模板：自定义常用命令，变量自动填充
- 选中路径按 L 直接打开文件/文件夹
- 粘完命令自动还原剪贴板，不影响你之前复制的内容

---

### 安装

```
git clone https://github.com/MenheraHann/TermKit.git
cd TermKit && make install
```

完全免费，MIT 开源。需要 macOS 13+。
